# Bayye Dashboard

**Bayye Dashboard** is an Admin Dashboard built using **Angular 18**, designed to manage the **Bayye B2B platform**, which connects suppliers with store owners.

The dashboard is responsible for managing:
- Users (Suppliers & Store Owners)
- Products
- Orders
- System settings
- Full integration with backend APIs

The frontend is fully developed using **Angular**, with seamless integration to **RESTful APIs** built with **.NET Core**.
